# Viewmodel.js
It is the script file used to adjust the weapon position, it works on userscript source file. You can change xyz coordinates with the help of [vscode](https://code.visualstudio.com/) to adjust your own weapon position. # ![JavaScript](https://img.shields.io/badge/javascript-%23323330.svg?style=for-the-badge&logo=javascript&logoColor=%23F7DF1E)
## Settings

```js
X: 0.11
Y: -0.50
Z: 0.65
```
